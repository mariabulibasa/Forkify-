# Forkify Project

A recipe application built as part of **Jonas Schmedtmann's JavaScript course on Udemy**.

🔗 Course: https://www.udemy.com/course/the-complete-javascript-course/?couponCode=KEEPLEARNING  
🌐 Live Demo: https://forkify-maria-bulibasa.netlify.app/

## Features

- 🔍 Search for recipes from an external API
- ➕ Add your own recipes
- 🔖 Bookmark favorite recipes (saved in local storage)
- 📄 Pagination for search results

## Tech Stack

- JavaScript (ES6+)
- Parcel (bundler)
- HTML & SCSS

## Notes

This project was developed following the course structure.
